# State of Charge | 충전 상태

↔ DoD Depth of Discharge<br>
배터리의 잔존 용량을 나타내기 위해 현재 사용할 수 있는 배터리 용량을 전체 용량으로 나눠 백분율(%)로 표시

#### SoC 측정법
1. **화학 측정법** : 액체 전해질에 직접 접근 가능한 축전지에만 사용 가능한 측정법, `배터리 전해질의 비중이나 산도(pH)를 측정`하여 계산
2. **전압 측정법** : `배터리 전압을 측정하고 방전 곡선과 대조`하여 계산
3. **전류 적산법(Coulomb Counting)** : `배터리의 전류를 측정한 후 시간에 따라 적분`하여 계산
4. **압력 측정법** : 니켈수소배터리의 잔존 용량을 측정할 때 사용 가능한 측정법, `배터리 내부의 압력을 측정`하여 계산
