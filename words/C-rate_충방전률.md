# C-rate | 충방전률

C-rate  = Current Rate [C(Capacity)] = 충방전 전류[A] / 배터리의 정격용량 값[Ah]

배터리를 충전 또는 방전시키는 속도를 나타내는 값이다.<br>
표준값은 1C

#### C-rate에 따른 소요시간과 가용 전류

|C-rate|소요 시간|가용 전류|
|---|---|---|
|0.1C|10H|0.1배|
|0.2C|5H|0.2배|
|0.5C|2H|0.5배|
|1C|1H|표준|
|2C|0.5H|2배|
|5C|0.2H|5배|
|10C|0.1H|10배|

=> C-rate가 높으면 방전속도가 빠르고 순간적으로 높은 전류를 요구하는 작업이 가능하다.

=> C-rate가 낮으면 고성능이 어려울 수 있다.
